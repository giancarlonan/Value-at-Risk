# Value-at-Risk (VaR)

A Jupyter Notebook implementation for calculating **Value-at-Risk (VaR)** for a random financial portfolio. VaR is a statistical technique used to measure the potential risk of loss in the value of a portfolio over a specified time period and confidence level.

## Features of the Notebook

- **VaR Calculation Methods**:
  - Monte Carlo Simulation
  - Quantile Regression + GARCH(1,1)
  - Quantile Regression + Neural Network (IN PROGRESS)
- Visualization of portfolio returns and risk metrics.
- Flexible inputs for portfolio configuration and risk assessment parameters.

## What is Value-at-Risk (VaR)?

Value-at-Risk (VaR) is a key risk management metric used by financial institutions to estimate the maximum expected loss of a portfolio with a certain level of confidence over a defined time horizon.

For example, a daily VaR of $1 million at 95% confidence implies that there is a 95% chance the portfolio will not lose more than $1 million in a single day.

## Applications

- **Risk Assessment**: Evaluate the potential loss in a portfolio under normal market conditions.
- **Decision Support**: Inform portfolio adjustments and hedging strategies.
- **Regulatory Compliance**: Meet financial risk disclosure requirements.
